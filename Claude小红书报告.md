## 一、平台算法机制

### 1.1 CES评分系统

小红书的推荐算法核心是**CES（Community Engagement Score）评分系统**，不同互动行为权重差异极大：

| 互动行为 | 权重分值 | 说明                      |
| -------- | -------- | ------------------------- |
| 关注     | 8分      | 权重最高，1个关注=8个点赞 |
| 评论     | 4分      | 真实评论价值高            |
| 转发     | 4分      | 社交裂变指标              |
| 收藏     | 1分      | 实用价值指标              |
| 点赞     | 1分      | 基础互动指标              |

**核心启示**：发布后2小时内引导评论和关注是撬动流量跃迁的关键杠杆。

### 1.2 八级流量池机制

| 流量池级别 | 曝光量    | 晋级条件              |
| ---------- | --------- | --------------------- |
| 初始池     | 200-500   | 互动率>5%，点击率>10% |
| 千人池     | 3000-5000 | 持续优质互动          |
| 万人池     | 1-5万     | CES评分优胜           |
| 十万池     | 10-50万   | 赛马机制胜出          |
| 百万池     | 100万+    | 顶级爆款              |

### 1.3 流量来源分布

- **发现页推荐**：约70%（算法推荐）
- **搜索流量**：约20-30%（增长最快，长尾效应强）
- **关注流量**：约5-10%（粉丝基础）

**关键发现**：超过65%的小红书流量来源于搜索行为，42%新用户第一天就使用搜索功能。

### 1.4 内容类型流量倾斜

- 视频笔记涨粉效率是图文的**3倍以上**
- 餐饮探店类视频占比超过80%
- 图文合集类内容收藏率更优，适合长期SEO

## 二、爆款判断标准

### 2.1 核心数据指标

| 指标        | 计算方式                 | 行业平均 | 优秀标准 | 餐饮参考        |
| ----------- | ------------------------ | -------- | -------- | --------------- |
| 打开率      | 阅读量÷曝光量           | 11%      | 15-20%+  | 封面吸引力      |
| 互动率      | (点赞+收藏+评论)÷阅读量 | 3%       | ≥10%    | 5-8%算优质      |
| 赞藏比      | (点赞+收藏)÷阅读量      | 3%       | ≥5%     | <2%无推流       |
| 收藏/点赞比 | 收藏量÷点赞量           | 0.3-0.5  | >0.8     | 探店攻略0.6-1.0 |

### 2.2 收藏/点赞比解读

| 内容类型   | 收藏/点赞比 | 说明           |
| ---------- | ----------- | -------------- |
| 菜谱教程   | 1.0-1.5     | 高实用价值     |
| 探店攻略   | 0.6-1.0     | 用户保存后打卡 |
| 纯吃播展示 | 0.2-0.4     | 娱乐性为主     |

### 2.3 潜力内容识别信号

| 时间节点        | 数据表现    | 判断               |
| --------------- | ----------- | ------------------ |
| 发布24小时内    | 互动数>200  | 有潜力             |
| 发布3天内       | 互动数>1000 | 爆款苗子           |
| 小眼睛:点赞收藏 | 10:1        | 已突破第一波流量池 |

### 2.4 账号权重影响

**低粉账号的爆款内容参考价值最高**：

- 1000粉账号获得1万点赞 = 绝对爆款，选题可直接复制
- 10万粉账号获得1万点赞 = 正常水平，包含粉丝惯性互动

### 2.5 评论质量判断

| 评论类型   | 特征                       | 价值           |
| ---------- | -------------------------- | -------------- |
| 询问型     | "这家店在哪？""人均多少？" | 购买意向强     |
| 经验分享型 | "我也去过，确实好吃"       | 增加可信度     |
| 具体追问型 | "不能吃辣怎么点？"         | 内容有应用场景 |
| 水军评论   | "太棒了""收藏了"           | 模板化，无价值 |

健康评论/点赞比例：**8%-15%**

### 11.2 短期热点vs长期趋势识别

#### 短期热点特征

- 热搜榜排名变化剧烈
- 笔记数量短时间激增
- 带有时间限定词（"刚刚""最新"）
- 生命周期：突发热点1-3天，节点热点7-14天，季节热点1-2个月

#### 长期趋势识别

- 搜索提示词长期稳定出现在下拉框
- 搜索量稳步上升而非暴涨
- 解决用户真实持续需求

#### 餐饮领域长期趋势

- 健康饮食（低卡/减脂/轻食）
- 性价比消费（平价美食/省钱攻略）
- 地域美食探索
- 一人食经济
- 亲子餐饮
- 情绪消费（治愈系/氛围感）

### 11.3 内容灵感库分类体系

#### 一级分类：内容类型

- 探店类 → 火锅/烧烤/西餐/日料...
- 教程类 → 食谱复刻/烹饪技巧...
- 测评类 → 单品/横评/避雷...
- 攻略类 → 城市/省钱/场景...

#### 二级分类：数据表现

- 爆款：点赞>1万
- 优质：点赞1000-1万
- 潜力：互动/粉丝比>5%

#### 必备标签维度

- 内容形式（图文/视频/合集）
- 互动级别
- 核心关键词
- 发布时间
- 适用场景（新店开业/节日营销/日常种草）
- 可复制性评级

#### 更新淘汰规则

- 每周新增10-20条爆款素材
- 每月更新关键词热度标签
- 淘汰发布超过6个月且未被参考的内容
- 淘汰互动数据低于同期平均值50%的内容
- 淘汰涉及过时热点的内容

### 11.4 竞品监控清单

#### 应监控的账号结构

- 头部竞品：5-10个（行业标杆）
- 腰部竞品：10个（差异化机会）
- 同城本地号：10-20个（本地流量密码）
- 素人爆款号：持续更新（低粉爆文规律）

#### 爆款复制公式

1. 筛选条件 → 互动数>粉丝数×5%
2. 提取元素 → 选题角度+标题结构+封面风格
3. 创新方向 → 换场景/换人群/换角度

### 11.5 内容发布前检查清单

- [ ] 标题包含核心关键词且前置，长度≤18字
- [ ] 正文开头100字包含核心关键词
- [ ] 关键词分布自然（每200字约1个），总计3-5次
- [ ] 标签符合3+2+1结构，共6个标签
- [ ] 已添加地理位置标注（本地餐饮必选）
- [ ] 封面图3:4竖图，有吸引力的文字或视觉焦点
- [ ] 发布时间在用户活跃期（工作日18:00-22:00，周末20:00-23:00）
- [ ] 评论区准备置顶补充说明（植入关键词）
