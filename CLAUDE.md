# Social Marketing 项目记忆

## 项目概述

社交媒体营销知识库项目，专注于餐饮连锁品牌的内容营销与爆品研发灵感采集。主要研究小红书、抖音、大众点评等平台的运营策略。

## 技术栈

- 文档格式：Markdown
- 版本控制：Git + GitHub
- 自动化采集：Python (计划中)
- 数据平台：千瓜、新红、灰豚 (第三方工具)

## 核心业务规则

1. **信息采集**：通过关键词组合搜索，采集爆款内容与产品灵感
2. **爆款判断**：点赞>500 + 赞粉比高（低粉高互动优先）
3. **内容发布**：遵循 CES 评分机制优化内容
4. **流量飞轮**：采集灵感 → 研发产品 → 内容营销 → 流量转化
