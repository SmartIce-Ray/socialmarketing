## 一、核心思维：新餐饮的认知觉醒

### 1.1 行业现状认知

| 传统逻辑（已失效） | 新逻辑（当下有效） |
|------------------|------------------|
| 选址好 + 口味好 = 生意好 | **认知差 + 执行力 = 竞争力** |
| 产品好自然有人来 | 没有流量 = 不符合年轻人需求 |
| 慢慢做口碑 | 极度内卷，一年倒闭数百万家 |

### 1.2 梭哈哲学（All-in）

**不是赌博，而是"全力精准打击"**

| 投入维度 | 具体要求 |
|---------|---------|
| 人力 | 团队全员投入，不能只靠一两个人 |
| 心力 | **老板必须亲力亲为**，甩手掌柜做不好 |
| 物力 | 预算充足，不要试探性投入 |
| 时机 | 看准机会再打，不盲目出击 |

### 1.3 认知红线

**拒绝自嗨**：不要只从老板角度觉得自家产品好

- 要从消费者角度看"买点"
- 如果产品没有流量，说明不符合当下需求
- 现在的"好吃" = **口味 + 颜值 + 情绪价值 + 出片率**

## 二、营销前置：门店基本功诊断

> **核心原则**：在做流量之前，必须先完成门店的自我诊断和优化，否则流量来了就是灾难。

### 2.1 选址与策略匹配

| 门店类型 | 营销策略 | 核心动作 |
|---------|---------|---------|
| 好商圈 + 好位置 | 不打折，做转化 | 吃自然流，提升客单价 |
| 好商圈 + 差位置 | **必须强营销** | 靠线上拉流，投流是必须的 |
| 社区店 | 靠熬，不做大营销 | 做复购，做私域，做口碑 |

### 2.2 菜单结构优化（扩客群）

**目的**：丰富客群，不能只做单一客群

| 产品类型 | 作用 | 示例特征 |
|---------|------|---------|
| 流量爆品 | 引流获客 | 视觉震撼 / 超低价格 |
| 地缘性产品 | 吸引本地人/游客 | 当地特色 / 非遗元素 |
| 成瘾性产品 | 提升复购 | 辣 / 咸鲜 / 重口味 |
| 甜品饮品 | 凑单增收 | 高颜值 / 适合拍照 |

### 2.3 场景与服务优化

#### 情绪价值打造

现在的消费者吃的是**综合体验**，不仅要好吃，还要：

- **好拍**：出片率高，适合发朋友圈/小红书
- **好玩**：切肉表演、跳舞、特殊上菜仪式
- **好聊**：有社交货币属性，值得分享给朋友

#### 服务红线管理

| 关键触点 | 动作要求 | 目的 |
|---------|---------|------|
| 门口排队 | 送小吃、陪聊、安抚情绪 | 降低流失率 |
| 桌边服务 | 主动桌访、询问满意度 | 及时发现差评苗头 |
| 差评处理 | 真诚沟通、解决问题 | **绝对禁止**花钱删差评或骚扰顾客 |

## 三、四大平台实战打法

> **营销三阶段**：工具使用 → 投流（确定性） → 文化传播（自传播）

### 3.1 大众点评：决策与转化

#### 平台定位

- **不是内容平台，是决策平台**
- 用户路径：搜 → 找 → 吃
- 核心价值：影响最终消费决策

#### 核心三要素

| 要素 | 说明 | 优化方向 |
|-----|------|---------|
| 访客量 | 流量入口 | 排名优化、推广通 |
| 评价体系 | 转化核心 | **橙V评价 > 数量堆积** |
| 团购套餐 | 交易转化 | 价格锚点、组合设计 |

#### 实战要点

**1. 评价质量 > 数量**

| 用户等级 | 价值 | 获取策略 |
|---------|------|---------|
| 橙V（Lv 5-8） | **一条顶十条** | 推出专属特价套餐（如58元双人餐） |
| 普通用户 | 基础累积 | 常规好评引导 |

**2. 差评控制**

- 排名不完全靠好评数，更看重**差评率**
- 必须通过真心服务和桌访来降低差评
- 差评出现后24小时内必须响应

**3. 推广通投放**

- 效果在下降，建议预算均摊
- 不要过度依赖，配合其他渠道使用

### 3.2 美团：下沉与销量

#### 平台定位

- 在下沉市场（三四线城市）非常有效
- 核心指标：**销量 = 排名**

#### 实战打法

| 策略 | 具体动作 | 目的 |
|-----|---------|------|
| 超低价引流 | 10元以下引流品 | 快速堆积销量基数 |
| 地推配合 | 发动员工线下推广 | 加速冷启动 |
| 人气榜冲刺 | 无论如何把销量刷上去 | 排名靠前 = 自然流量 |

### 3.3 小红书：种草与搜索截流

#### 平台定位

| 特性 | 说明 |
|-----|------|
| 审美红利 | 颜值即正义，视觉决定点击 |
| 长尾流量 | 优质笔记6个月后仍有流量 |
| 搜索入口 | 60%流量来自主动搜索 |
| **第一阵地** | 目前餐饮营销最重要的平台 |

#### 内容策略

**1. 漂亮饭法则**

- 封面图决定点击率
- 2024-2025年流行色：**蓝色、紫色**
- 出片率 > 产品本身

**2. KFS矩阵**

| 模块 | 说明 | 预算占比建议 |
|-----|------|------------|
| K（KOL/KOC） | 达人铺量 | 40% |
| F（Feeds） | 信息流投流 | 30% |
| S（Search） | 搜索卡位 | 30% |

#### 投流策略（聚光平台）

**1. 赛马机制**

```
核心逻辑：不平均分配，放大爆款

步骤：
1. 多篇笔记同时发布
2. 观察24-48小时自然数据
3. 数据好的笔记 → 追加投流（薯条/信息流）
4. 数据差的笔记 → 放弃，不浪费预算
```

**2. 搜索词布局**

| 词类型 | 示例 | 作用 |
|-------|------|------|
| 品类词 | "杭州火锅" | 精准获客 |
| 场景词 | "约会餐厅""生日聚餐" | 场景截流 |
| 竞品词 | 竞争对手品牌名 | 抢夺竞品流量 |

**3. 投流红线**

> **一定要投流！** 现在的自然流很难，要用钱买确定性，不要赌运气。

### 3.4 抖音：爆发与直播

#### 平台定位

| 特性 | 说明 |
|-----|------|
| 泛流量 | 覆盖面广，人群不精准 |
| 爆发力强 | 短时间内可冲极高曝光 |
| 衰退快 | 热度周期短，需持续投入 |
| **适用场景** | 新店开业、大促活动 |

#### 直播打法

**1. 人设IP打造**

- **老板/老板娘亲自下场**
- 打造真实人设，不要端着
- 示例话术："我是个做饭的，不会说话，但东西真材实料"

**2. 话术逻辑**

| 方法 | 说明 | 示例 |
|-----|------|------|
| 幼儿园教法 | 互动、提问，不要单向输出 | "家人们觉得这个怎么样？" |
| 胸口碎大石 | 先聚人气再卖货 | 前30分钟不卖货，纯互动 |
| 痛点营销 | 挖掘顾客怕什么 | 怕踩雷 → 强调老板人品 |

**3. 场景化**

- **实景直播（后厨/明档）>> 绿幕直播**
- 真实感 = 信任感 = 转化率

## 四、营销节奏与时间线

> **核心原则**：有1有2还有3，不能打完一波就停

### 4.1 第一波：出圈（以快制快）

| 维度 | 要求 |
|-----|------|
| 时间 | 集中在开业前3-5天 |
| 动作 | 达人密集发布 + 投流放大 |
| 目的 | 制造"全城都在吃"的假象 |
| 效果 | 冲上热门榜单 |

**关键策略**：

- 利用羊毛党/排队制造人气
- 这期间不赚钱，但赚人气和声量
- **红线**：运营和服务必须扛住，避免崩盘

### 4.2 第二波：破圈（做对的事）

| 维度 | 要求 |
|-----|------|
| 时间 | 开业后1-2周 |
| 动作 | 筛选爆款追投 + 换客群 |
| 目的 | 从虚假繁荣转向真实生意 |
| 效果 | 提升转化率和客单价 |

**关键策略**：

- 第一波数据好的内容 → 追加投流
- 客群转换：羊毛党 → 家庭客/商务客
- 开始做精准人群的内容

### 4.3 第三波：扩圈（持续经营）

| 维度 | 要求 |
|-----|------|
| 时间 | 开业后1个月起 |
| 动作 | 沉淀品牌资产，做文化营销 |
| 目的 | 形成自传播 |
| 效果 | 顾客主动拍照分享 |

**三大文化方向**：

| 文化类型 | 说明 | 示例 |
|---------|------|------|
| 产品文化 | 产品背后的故事 | 食材来源、独家工艺 |
| 空间文化 | 门店环境的打卡点 | 网红墙、特色装置 |
| 表演文化 | 现场体验的仪式感 | 上菜表演、互动环节 |

## 五、避坑指南与红线警告

### 5.1 战略层避坑

| 错误做法 | 正确做法 |
|---------|---------|
| 盲目模仿大牌 | 小店要算账，讲究"确定性" |
| 依赖代运营/甩手掌柜 | **老板必须下场**，决心决定生死 |
| 旺季做活动牺牲利润 | **淡季取势，旺季获利** |
| 产品不行强做营销 | 营销只能加速，不好吃 = 加速死亡 |

### 5.2 执行层红线

| 红线行为 | 后果 |
|---------|------|
| 花钱删差评 | 平台处罚 + 口碑崩塌 |
| 骚扰顾客改评价 | 法律风险 + 负面传播 |
| 试探性小额投入 | 钱花了没效果，不如不投 |
| 平均分配达人预算 | 浪费资源，没有爆款效应 |

### 5.3 活动设计原则

**做活动的目的是拉新和好评，不是亏损**

| 原则 | 具体做法 |
|-----|---------|
| 算清账 | 利用高客单价 + 叠加使用规则来保本 |
| 有目的 | 霸王餐/低价券 = 换好评/换曝光 |
| 有节奏 | 淡季做活动，旺季收利润 |

### 5.4 未来趋势

> **拥抱AI和算法**：谁先理解算法，谁就先吃到红利

- 高德各种榜单算法
- 小红书推荐算法
- 抖音流量分发逻辑

## 六、执行清单与SOP

### 6.1 营销启动前检查清单

- [ ] 门店选址与策略已匹配
- [ ] 菜单结构已优化（流量品+利润品+地缘品）
- [ ] 出片场景已打造（至少3个打卡点）
- [ ] 服务SOP已培训（排队安抚+桌访话术）
- [ ] 预算已准备充足（不做试探性投入）
- [ ] 老板已确认亲自下场

### 6.2 平台优先级（按重要性排序）

| 优先级 | 平台 | 核心动作 |
|-------|------|---------|
| 1 | 小红书 | KFS矩阵 + 赛马投流 |
| 2 | 大众点评 | 橙V评价 + 差评控制 |
| 3 | 抖音 | 老板直播 + 开业爆发 |
| 4 | 美团 | 销量堆积（下沉市场优先） |

### 6.3 三波节奏时间表

```
【开业前5天】
- 达人内容集中发布
- 投流预算集中投放
- 制造排队效应

【开业第1-2周】
- 筛选爆款追投
- 客群转换动作
- 服务承接验证

【开业1个月后】
- 文化内容沉淀
- 自传播机制建设
- 长红模式确立
```

### 6.4 每日/每周动作清单

#### 每日必做

- [ ] 查看各平台数据（小红书/点评/抖音）
- [ ] 回复评论和私信
- [ ] 处理差评（24小时内响应）
- [ ] 记录今日出片素材

#### 每周必做

- [ ] 数据复盘（哪些内容爆了，为什么）
- [ ] 投流调整（追加爆款，砍掉差的）
- [ ] 达人沟通（下周内容规划）
- [ ] 服务复盘（差评原因分析）

## 总结：全域营销核心公式

```
梭哈式资源投入（资金 + 心力）
        ×
全域平台精细化运营（投流 + 搜索截流）
        ×
情绪价值产品包装（社交货币 + 出片率）
        ×
门店优质服务承接（体验闭环）
        ↓
    网红店 → 长红店
```
